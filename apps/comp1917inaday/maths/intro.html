unsigned and long are keywords that can be used to prepend certain types of variable as they are declaired to give them certain properties. Some examples of this can be found later. It is important to understand how each type of variable stores data in memory. char
This is just a single byte unsigned integer. That is, it is always positive and stores an integer between 0 and 255. They can be interpreted as a character when it matters (ie when dealing with standard input and output) but can also be treated as an integer. The character that a char variable is interpreted as is the ascii character corresponding to the integer value stored. int
The first bit of an integer represents its sign. A 1 represents a negative number and a 0, a positive number. In order to do mathematical operations on negative binary numbers, a method called 2's compliment is used. Negative numbers are stored in 2's compliment, and can then be added to positive numbers.
